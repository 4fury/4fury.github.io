Pisi Linux -- дистрибутив, построенный на технологиях, разработанных в рамках пректа Pardus, отличительная особенность которых в активном использовании языка python для установки, загрузки и настройки системы, упаковки и конфигурирования пакетов. Изначально целью проекта было сохранение наследия оригинального Pardus в надежде, что разработка последнего будет возобновлена. Сейчас Pisi Linux все больше позиционирует себя как самостоятельный дистрибутив для десктопов без серверной редакции. В настоящий момент ключевые компоненты системы переписываются на python3. Релиз Pisi Linux 3 ожидается в 2020 году.
